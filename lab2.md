Then we can see the syscall instruction:
```c
void
syscall(void)
{
  int num;
  struct proc *p = myproc();

  num = p->trapframe->a7;
  if(num > 0 && num < NELEM(syscalls) && syscalls[num]) {
    // Use num to lookup the system call function for num, call it,
    // and store its return value in p->trapframe->a0
    p->trapframe->a0 = syscalls[num]();
  } else {
    printf("%d %s: unknown sys call %d\n",
            p->pid, p->name, num);
    p->trapframe->a0 = -1;
  }
}
```
1. a0 and a1 is our exec arguments(for initcode.s)
2. a7 is our system call number(that is , i mean it's for exec,  SYS_exec)
3. its return value is in `p->trapframe->a0`
